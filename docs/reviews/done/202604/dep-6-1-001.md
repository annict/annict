  ```yaml
  # 現在のコード（L133-134）
  - pkg: github.com/annict/annict/go/internal/query
    desc: "HandlerはQueryに直接依存できません。Repositoryを経由してください。"
  ```

  **修正案**:

  ```yaml
  - pkg: github.com/annict/annict/go/internal/query
    desc: "HandlerはQueryに直接依存できません。UseCase経由でアクセスしてください。"
  ```
