**問題点・改善提案**:

- **[コメントのガイドライン]**: handler-layer セクションのコメント L127 `# QueryへのアクセスはRepositoryを経由` が現在のルールと矛盾している

  Handler は Repository にも直接アクセスできなくなったため、「Repositoryを経由」という記述は開発者を誤解させる可能性がある。L128 で新しいコメントが追加されているが、L127 の古いコメントが残っているため、矛盾が生じている。

  ```yaml
  # 現在のコード（L126-128）
  # Handler層のルール: HTTPリクエスト処理
  # QueryへのアクセスはRepositoryを経由
  # データアクセスはUseCase経由で行い、RepositoryやValidatorに直接依存しない
  ```

  **修正案**:

  ```yaml
  # Handler層のルール: HTTPリクエスト処理
  # データアクセスはUseCase経由で行い、Repository・Validator・Queryに直接依存しない
  ```
