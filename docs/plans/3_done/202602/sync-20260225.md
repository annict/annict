# プロジェクト間ファイル同期 2026/02/25

## 概要

Annict、Mewst、Wikino の 3 プロジェクト間でガイドラインや設定ファイルの差分を調査し、同期を行う。

## 設計

### 差分なしのファイル

以下のファイルは 3 プロジェクト間で実質的な差分がなかった：

- `go/docs/handler-guide.md`
- `go/docs/i18n-guide.md`
- `go/docs/validation-guide.md`
- `rails/.prettierrc`
- `rails/eslint.config.mjs`

- `.claude/commands/review.md`
- `.claude/commands/sync.md`
- `docs/specs/template.md`

### プロジェクト固有の差分（対象外）

以下の差分はプロジェクト固有の事情によるものであり、同期対象外とする：

- **バージョン番号**: Go, Ruby, Node.js, PostgreSQL, pnpm 等の各プロジェクト固有のバージョン
- **プロジェクト固有の機能/インフラ**: Annict の imgproxy/R2/Redis/Stripe/Playwright/GraphQL、Mewst の Caddy/DaisyUI、Wikino の Worker/ドメインID型/SpaceID/CodeMirror/libvips 等
- **プロジェクト固有の CI 環境変数・ターゲット**: seed, graphql-dump, test-setup, worker 等
- **プロジェクト固有のパッケージ**: Playwright 関連パッケージ、Redis tools、Stripe CLI 等
